# Pcontum-3D
OpenGL ile yapılmış bir program.
## Özellikler: ##
- Freeglut kullanıldı
- Mouse ile kontrol
- Klavye ile kontrol
- %70 Türkçe
- Modelleri değiştirebilme
# Kurulum
Güncelle:
```sudo apt-get update```
Kütüphaneleri yükle:
```sudo apt-get install mesa-common-dev libglu1-mesa-dev freeglut3-dev```
# Derle
```make```
# Çalıştır
```./Pcontum3D``` veya ```Pcontum3D```'ye çift tıkla
